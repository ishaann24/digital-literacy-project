Project Overview (Digital Literacy Portfolio)
This project aims to build important digital skills by placing you in the role of a Student Digital Ambassador. In this role, you will help others understand how to use digital tools efficiently, responsibly, and professionally. The project includes five hands-on tasks, each connected to a specific module, and all your work will be compiled in a GitHub repository along with a final report.

Task Breakdown

Task 1: Digital Literacy Infographic
Design an engaging infographic using tools such as Canva or Prezi. The infographic should explain digital literacy and cover topics like student tools, safe internet practices, or proper email usage.
Focus: Creativity and awareness.

Task 2: Student Digital Portfolio
Develop your professional online presence by creating or improving profiles on platforms like GitHub and LinkedIn.
Focus: Career development and personal branding.

Task 3: Coding & Collaboration Platforms
Explore practical tools by:

Solving a beginner-level problem on a coding platform
Creating a digital literacy quiz using Google Forms
Focus: Hands-on learning and collaboration tools.

Task 4: Email Etiquette & Online Behavior
Write professional emails and prepare a checklist outlining appropriate and inappropriate behavior on social media.
Focus: Effective and respectful digital communication.

Task 5: Cybercrime Awareness
Create a case study on a cybercrime topic (such as phishing or UPI fraud) and design a checklist for prevention.
Focus: Online safety and awareness.

Final Objective

By completing all tasks, you will:

Develop responsible and informed digital habits
Build a strong professional online identity
Gain experience with tools commonly used in education and the workplace
